## Status

None

## Syntax

`PREUATT:x`

## Parameters

-   Variables Used: Number (Unarmed Attack Bonus number).



What it does
------------

Number indicates the minimum unarmed base attack bonus.

Example
-------

`PREUATT:4`

This would apply only if the PC had minimum unarmed attack bonus of 4.
