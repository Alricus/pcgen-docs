## Status

None

## Syntax

`XPPENALTY:x`

## Parameters

-   x: Text (YES or NO).



What it does
------------

Specifies whether a XP Penalty should be applied.

Example
-------

`XPPENALTY:YES`

A Penalty to XP will be applied.

Where it is used
----------------

CLASSTYPE Line.
