## Status

New 5.6.1

## Syntax

`ALLOWAUTORESIZE:x`

## Parameters

-   x: Boolean (Y or N).



What it does
------------

This tag is used in the <span class="lstfile"> gameMode/miscinfo.lst
</span> file to identify whether PCGen should allow the auto-resizing of
equipment.

Example
-------

`ALLOWAUTORESIZE:Y`

This gameMode will automatically resize equipment when purchased
