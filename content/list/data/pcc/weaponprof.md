## Status

None

## Syntax

`WEAPONPROF:x`

## Parameters

-   x: Text (File path)



What it does
------------

Tells PCGen to load the weapon proficiencies list file specified.

Example
-------

`WEAPONPROF:fooweapprofsimple.lst`

Load the weapons proficiencies *fooweapprofsimple.lst* file.
